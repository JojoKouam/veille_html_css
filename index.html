Méthodes d'apprentissage


La Méthode classique


De façons classique l'apprentissage se fait par:


Les cours interactifs comme FreeCodeCamp - Idéal pour l'apprentissage pratique.


Les ateliers et les groupes de travail comme Codecademy - Idéal pour les débutants.


Livres tel que HTML et CSS : Design and Build Websites par Jon Duckett - Un livre visuel et facile à suivre.


Suivre des tutoriels YouTube.


Methode moderne


L'apprentissage peux être beaucoup plus fun on peut apprendre par le jeux comme:


Flexbox Froggy(Apprendre le positionnement en CSS)


CSS Grid Garden(Maitriser le CSS Grid)


CodeCombat(Apprendre le JavaScript et le HTML/CSS)
